Markdown，文本修饰语言，用特殊符号修饰正文效果

# 标题修饰符\#
# （一级标题）
## （二级标题）
### (三级标题)

## 正文内容
输入正文内容，但是如果需要换行，用\<br\>标签

## 修饰正文
*一段测试文本*
**一段测试文本**
***一段测试文本***
~~一段测试文本~~

## 分割线
用\-\-\- 表示分割线

---

## 引用\>表示
> 一级引用
> > 二级引用
> > > 三级引用
> > >
> > > > 四级引用

## 列表修饰符
### 无序列表 \*
* 123
  * 1
  * 2
  * 3
* 456
  * 4
  * 5
  * 6

## 有序列表 1.
1. 编程语言
   1. C
   2. C++
   3. Python
2. 计算机
   1. 计算机网络
   2. 操作系统
   3. 计算机组成原理
   4. 数据结构与算法

### 表格
字母|数字|排行
:--|:--:|--:
--\|:--:\|--:表示这个表格第一列靠左对齐，第二列居中，第三列靠右||
A|123|1
B|456|2
C|789|3
